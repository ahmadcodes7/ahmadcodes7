## 💡 About Me

<img align="right" src="https://media.giphy.com/media/M9gbBd9nbDrOTu1Mqx/giphy.gif" width="220"/>

I'm a **Network Engineer** with ~1 year of hands-on experience. My work sits at the intersection of enterprise routing, switching, and network security — and I'm constantly pushing deeper into each layer.

I build **35–40 router topologies** in GNS3 and EVE-NG, analyze traffic at the byte level using Wireshark and tcpdump, and troubleshoot everything from OSPF convergence to IPsec VPN mismatches. Every lab is a chance to go from *"I know the concept"* to *"I've broken it and fixed it."*

**👨‍💻 Currently working on...**

- IDS/IPS, ISE
- Packet-level analysis with Wireshark & tcpdump — DNS internals, DHCP edge cases
- Firewall rule validation, ACL design, and CCNP Security track labs
- Python scripting for basic network automation and device interaction

**🎯 Goal:** Specialize in Network Security — using a strong infrastructure foundation to protect and harden enterprise systems.
